# IPL Data Analysis and Visualization

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on the Indian Premier League (IPL) dataset using Python. The goal is to extract meaningful insights from match data and visualize patterns using powerful data analysis and visualization libraries.

The project demonstrates real-world data analysis skills including data cleaning, transformation, and visualization.

---

## Technologies Used

* **Python**
* **Pandas** – Data manipulation and analysis
* **Matplotlib** – Data visualization
* **Seaborn** – Advanced statistical visualizations

---

## Dataset

The dataset contains IPL match details such as:

* Teams
* Match results
* Winning margins
* Top scorers
* Match venues

---

## Key Analysis Performed

* Analysis of team performance across seasons
* Highest winning margins (runs & wickets)
* Top scoring players and highest individual scores
* Match outcome trends (batting vs bowling first)
* Filtering and sorting data using Pandas

---

## 📊 Data Visualization

Various visualizations were created to understand the data better:

* Bar plots for team performance
* Count plots for match outcomes
* Heatmaps for correlation analysis
* Distribution plots for score patterns

---

## Key Insights

* Certain teams consistently perform better across seasons
* Winning margin analysis highlights dominant performances
* Toss decision impacts match outcomes in some cases
* High-scoring matches show interesting scoring trends

## How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ipl-data-analysis-and-visualization.git
   ```

2. Install required libraries:

   ```bash
   pip install pandas matplotlib seaborn
   ```

3. Run the Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
